При команде git init - создаётся системный файл (внутренняя папка).

- При создании первого файла нужно сообщить Git об этом.

- git status покажет Мастер ветку.

- Отслеживать файл: git add <name file...>

- Перестать отслеживать: git rm --cached <name file...>

- Подключить для отслеживания сразу все файлы: git add . (точка в конце)
